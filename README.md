# Flutter Navigation App

A basic Flutter application that demonstrates navigation between multiple screens using `Navigator.push()` and `Navigator.pop()`.

##  Features

* Two screens (First Screen & Second Screen)
* Navigate to a new screen using a button
* Return back to the previous screen
* Simple and clean UI using Material Design

##  Tech Stack

* Flutter
* Dart

### Prerequisites

* Flutter SDK installed
* Android Studio / VS Code
* Emulator or physical device

##  How It Works

* The app starts with `FirstScreen`.
* `Navigator.push()` is used to move to `SecondScreen`.
* `Navigator.pop()` returns back to the previous screen.
* Each screen is built using a `StatelessWidget`.
